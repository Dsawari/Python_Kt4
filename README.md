## User Management System

A simple Python project demonstrating object-oriented programming (OOP) concepts such as:
- Classes and inheritance
- Encapsulation
- Properties (@property)
- Getters and setters
- Operator overloading
- Class attributes

## Features
- Create regular users and super users
- Count created users
- Hide passwords
- Prevent login changes
- Compare users by grade
- Demonstrate inheritance with SuperUser

## Technologies Used
- Python 3

## Concepts Demonstrated
- OOP in Python
- Inheritance
- Encapsulation
- Properties and setters
- Magic methods (__lt__, __gt__, __eq__)
- Class variables
